Process

How It Works

Three simple steps from image upload to AI-assisted classification in seconds.

01
📷

Upload Your Image

Drag and drop or select a JPEG, PNG, or BMP photo of the skin lesion. The system validates format and confirms skin is present before proceeding.

02
🧠

AI Analysis

Your image is preprocessed and passed through MobileNetV2 with Squeeze-and-Excitation attention, producing a confidence score across 7 classes.

03
🔥

Grad-CAM Explanation

A Grad-CAM++ heatmap overlay highlights exactly which regions the model focused on — making every decision transparent and explainable.

Capabilities

Key Features

Built with security, privacy, and explainability at its core.

🔥

Grad-CAM++ Heatmaps

Visual explanation of every prediction via the SE-attention multiply layer — showing where the model looked, not just what it predicted.

🔐

JWT Authentication

Secure access and refresh token system with bcrypt password hashing, rate limiting, and automatic session management.

🛡️

Zero Medical Data Storage

No images, predictions, or diagnoses are ever stored. Your analysis is processed in-memory and discarded immediately after the response.

🧬

SE Attention Architecture

MobileNetV2 enhanced with a Squeeze-and-Excitation block that recalibrates channel-wise features for improved lesion detection.

🚫

Skin Presence Detection

YCbCr-based pre-inference gate rejects non-dermatological images using colour space analysis and morphological coherence checking.

⚡

Fast REST API

Built with FastAPI and async processing. MVC architecture with Repository, Factory, Strategy, Observer and Singleton design patterns.

Classification

7 Conditions Detected

Trained on the HAM10000 dataset — dermatoscopic images from leading medical institutions.

Pre-cancerous

Actinic Keratosis

Rough, scaly patch from sun exposure. May progress to squamous cell carcinoma if untreated.

Malignant

Basal Cell Carcinoma

Most common skin cancer. Slow-growing and rarely spreads. Highly treatable when caught early.

Benign

Benign Keratosis

Non-cancerous seborrhoeic growths. Waxy, stuck-on appearance. No treatment usually needed.

Benign

Dermatofibroma

Common benign skin nodule, typically on the legs. Firm and slightly raised. Generally harmless.

Malignant

Melanoma

Most serious skin cancer. Early detection is critical — seek professional advice immediately.

Benign

Melanocytic Nevi

Common moles — benign melanocyte proliferations. Monitor for changes in size, shape, or colour.

Vascular

Vascular Lesion

Blood vessel abnormalities including haemangiomas. Usually benign but warrant clinical assessment.

Important Notice

About This Tool

Please read before using DermAI.

🚨

NOT a Medical Diagnosis

DermAI is an educational and research tool developed as part of an academic dissertation. It demonstrates AI-assisted dermatology classification and explainability techniques. Predictions can and will be inaccurate. This tool must not be used as a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified dermatologist for any skin concerns.

🎓 Academic Context

Built as a final-year dissertation project demonstrating full-stack AI application development with MVC architecture, GoF design patterns, and explainable AI techniques.

📊 Dataset

The model was trained on the HAM10000 dataset — 10,000+ dermatoscopic images from the Medical University of Vienna and Skin Cancer Institute Queensland.

🔒 Privacy

Zero data retention. No images, diagnoses, or personal health information are ever stored on any server. All processing is ephemeral and in-memory only.

⚙️ Technology Stack

FastAPI · MobileNetV2 + SE Attention · Grad-CAM++ · PostgreSQL · JWT Auth · Python 3.13 · TensorFlow 2.x · Vanilla JS frontend.
